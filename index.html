Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The p element or also known as The Paragraph element is were you write a block of text that is shown on your website.
  • The title element defines the title of your webpage. It is requiered and is extremely important for Search Engine Optimization.
  • Opening and closing tags are often represented by angle brackets with associated tag names contained inside.The closing tag begins with a forward slash before the tag name.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Incline CSS stlye attribute allows you to change the color of a single h1 heading.
  • Internal CSS style sheet can be embeded directly into the HTML file by use style element inside the head element.
  • External CSS style sheet we have to link the file by using the link element and placing it inside the head element which is inside the HTML file.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git checkout main: switches to your main branch
  • git checkout /branch-name: switches you to the branch you want to work on
